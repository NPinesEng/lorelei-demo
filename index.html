About LoRELEI

LoRELEI (Long-Range Environmental Location and Event Intelligence) is a GPS tracking system designed specifically for outdoor endurance events in areas without cellular coverage.

Using LoRa mesh networking technology, LoRELEI provides real-time tracking of runners, cyclists, and other participants across remote terrain where traditional GPS tracking solutions fail. The system supports up to 100 concurrent runners with position updates every 20 seconds.

Developed by Nein Pines Engineering for competition shooting events and endurance races, LoRELEI transforms race management from reactive (manual timing, volunteer spotters) to proactive (automated geofence detection, real-time alerts).

Key Features

No Cellular Required

LoRa mesh networking provides multi-mile coverage in remote areas where cell phones do not work. Perfect for backcountry events.

Real-Time Tracking

Position updates every 20 seconds with automatic geofence detection for starts, stages, and finishes.

Multi-Day Battery

Compact GPS trackers provide 24+ hours of continuous operation on a single charge. IP67 waterproof enclosures.

Automated Scoring

Automatic checkpoint detection and timing without manual intervention. GPS-based timestamps eliminate human error.

Safety Monitoring

Alarm zone geofences detect off-course runners immediately. Idle detection alerts for potentially stuck participants.

Live Leaderboards

Real-time results display for spectators and officials. Export to CSV for official scoring.
